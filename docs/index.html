WaferMap FBM 프로젝트 가이드

이 프로젝트의 본질은 fail bit map을 보고 defect pattern을 위치와 family 단위로 인식하고, 수치화하고, 비슷한 wafer를 찾을 수 있는 Python 기반 AI 파이프라인을 만드는 것이다.

프로젝트 개요 데이터 흐름 누끼 asset 실제 PNG 실행 모델 전략 검증 기준 용어집

한 문장 정의

실제 FBM에서 defect pattern mask를 모으고, 그 mask로 라벨이 명확한 합성 wafer를 만들고, U-Net 계열 multi-label segmentation 모델과 embedding retrieval을 붙여 “어디에 어떤 불량이 얼마나 있는지”를 계산한다.

현재 프로젝트는 단순 리포트 생성기가 아니라 defect mask asset builder, synthetic dataset generator, segmentation training, 유사 wafer retrieval을 하나의 흐름으로 묶는 작업이다.

전체 흐름

실제 FBM 입력

제품별 raw PNG 또는 semantic npz를 준비한다. 기본 위치는 data/raw/<product>/*.png다.

누끼 asset 생성

editor에서 defect 영역을 따고 family별 폴더에 저장한다. 예: data/masks/blob/, data/masks/ring/.

합성 데이터 생성

mask asset을 wafer 배경에 합성한다. 이때 위치, family, 면적, severity 변화가 metadata로 남는다.

딥러닝 학습

multi-channel mask를 정답으로 U-Net 계열 segmentation 모델을 학습한다.

수치화와 검색

모델 출력에서 family별 면적, 중심, radial/angle 위치, embedding vector, top-k similar wafer를 만든다.

전문가 피드백

실제 wafer 결과를 보고 틀린 family, 누락, 위치 오류를 기록한 뒤 generator와 model을 보정한다.

파일을 어디에 두는가

목적기본 위치설명
실제 FBM raw PNG data/raw/<product>/*.png Grade 0~7 grayscale 이미지. 사내 파일 경로를 직접 넣어도 된다.
제품 geometry data/raw/product_geometry.json 제품별 chip block, grid, actual net die 정보를 둔다.

모델 산출물 outputs/models/<run>/ checkpoint, metrics, prediction preview, embedding index를 둔다.

실제 PNG batch 실행

실제 FBM PNG를 제품별 폴더에 넣은 뒤 아래처럼 실행한다.

python scripts/analyze_png_raw_folders.py `
  --raw-root data/raw `
  --geometry-json data/raw/product_geometry.json `
  --out-dir outputs/reports/real_png_batch `
  --cpu-model outputs/pre_real_readiness/models/fbm_cpu_encoder_model.npz

주요 산출물은 아래와 같다.

모델 정의

왜 U-Net인가

FBM 문제는 이미지 전체 분류보다 병변 segmentation에 가깝다. “불량이 있다”보다 “어느 위치에 어떤 모양으로 있다”가 중요하므로 pixel-level mask를 출력하는 U-Net 계열이 baseline으로 적합하다.

왜 multi-label인가

한 wafer에는 ring과 blob, edge와 scratch가 동시에 있을 수 있다. 따라서 softmax 단일 클래스보다 family별 sigmoid channel을 두는 multi-label segmentation이 맞다.

왜 embedding도 필요한가

segmentation은 위치와 family를 주고, embedding은 유사 wafer 검색을 담당한다. 최종적으로는 mask feature와 encoder embedding을 함께 써서 top-k similar wafer를 찾는다.
